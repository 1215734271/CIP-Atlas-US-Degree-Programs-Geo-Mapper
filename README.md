# CIP-Atlas: US Degree Program GeoMapper

![Map Preview Placeholder](https://img.shields.io/badge/Visualization-GeoMap-blue?style=flat-square) ![Python Version](https://img.shields.io/badge/Python-3.8+-brightgreen?style=flat-square) ![IPEDS Data](https://img.shields.io/badge/Data-IPEDS_2024-orange?style=flat-square)

**CIP-Atlas** is an automated data visualization pipeline designed to map the geographical distribution of academic programs across United States higher education institutions. By leveraging official IPEDS (Integrated Postsecondary Education Data System) data and specific CIP (Classification of Instructional Programs) codes, this tool generates interactive, presentation-ready heatmaps and scatter plots in HTML format.

## 🎯 Features

-   **Data-Driven Accuracy**: Utilizes official IPEDS Completion (`c2024_a.csv`) and Institutional Characteristics (`hd2024.csv`) datasets.
-   **Precision Targeting**: Queries institutions based on exact CIP Codes, ensuring you map specific degree-granting programs, not just broad university departments.
-   **Interactive Visualization**: Automatically generates high-quality interactive HTML maps using `pyecharts`. Maps feature state-level heat distributions and precise university coordinate scatter points.
-   **Highly Extensible**: The pipeline is designed to be easily modified for *any* CIP code to generate new program maps instantly.

## 🗂 Data Pipeline Workflow

1.  **Extract**: `US_major_Query.py` filters the IPEDS Completions dataset using a specified `CIPCODE` to find institutions granting degrees in that field.
2.  **Transform**: It joins these results with institutional data to append geographic coordinates (Longitude/Latitude) and state information.
3.  **Visualize**: `US_major_university_map.py` processes the clean data to compute state-level statistics and renders an interactive geographical map.

## 🚀 Quick Start

### Prerequisites
Make sure you have the required Python libraries installed:
```bash
pip install pandas seaborn pyecharts
```

*(Note: The raw IPEDS data files `c2024_a.csv` and `hd2024.csv` must be present in the directory to run queries for new programs.)*

### Generating a New Map
1. Open `US_major_Query.py`, modify the `ai_cipcode` and `ai_major_name` variables to your target program, and run the script to generate a `.csv` dataset.
2. Open `US_major_university_map.py`, update the `major_name` and `major_abbr` variables, and run the script to output the `.html` map.
